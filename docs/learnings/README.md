## Tips para Buenas Notas

1. **Usa tus propias palabras**: No copies y pegues, reescribe con tu entendimiento
2. **Incluye ejemplos**: El código habla más que las palabras
3. **Relaciona conceptos**: Conecta nuevos conocimientos con lo que ya sabías
4. **Actualiza regularmente**: Vuelve a tus notas y mejóralas con nuevo entendimiento
5. **Sé específico**: "useState" es mejor que "React stuff"
